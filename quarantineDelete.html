Mails not getting deleted from O365 Quarantine
20-Aug-24

You go to your Office365 Quarantine, you delete an email, making sure to tick the box to permanently delete the mail. You then go back to the main Quarantine page and find the deleted mail still there. If you try and select the mail it says it has already been deleted.

This seems to have occurred for a number of O365 users over the last few years, alas a permanent fix does not seem to be forthcoming from Microsoft.
