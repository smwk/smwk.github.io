Set Locale for various Apps
12-Mar-23

No matter what previous settings have been set you will nearly always end up with a US locale for your app, the following is a list of ways to change the various apps:
AppSetting
Teams ClientNot really possible to do remotely with PowerShell. The package needs to be deployed with the language set
Teams files in SharePoint The SharPoint template can be edited and existing sites updated.
OneDrive for BusinessNo possible to edit the SharePoint template, but possible to change existing sites with PowerShell.
ExchangeOnline mailboxCan be set with PowerShell.
Azure AD accountCan be set with PowerShell.
Windows 10No PowerShell commands, so language needs to be set in deployment image.
Windows 11Does have PowerShell commands to set the language - Set-WinSystemLocale.
