Installing and configuring PHPMailer
25-Jun-24

PHPMailer has a number of advantages over the built-in mail() function, most importantly for me is the ability to call attached images inline.

While I primarliy use PowerShell for mail sending scripts, it's always a good idea to have a backup and it's fun to learn new tricks.

The preffered method for installing is to use Composer, but unfortunetly I couldn't use this as it doesn't support an authenticated web proxy. The other way to install is to download a Zip file. This should be unpacked to the PHP web root, so with IIS this is typcially c:\intersrv\wwwroot\.
